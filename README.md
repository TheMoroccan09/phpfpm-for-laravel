# PHPFPM-FOR-LARAVEL
A docker image to create a laravel projects

### Built with 
 * PHP v7.3.4
 * Composer 1.10.8
 * Extentions:
  * OpenSSL
  * PDO
  * Mbstring 
  * Tokenizer
  * Ctype
  * JSON
  * BCMath
  * gd
  * Others

### How to run it 
```
	docker run --name php themoroccan09/phpfpm-for-laravel
```
